# 🚀 GritX

**GritX** is a premium student marketplace app built with Flutter where users can **buy, sell, and connect locally**.

---

## ✨ Features

* 🔐 Google Sign-In & Email Authentication
* 📍 Location-based access (GPS integration)
* 🛒 Buy & Sell products easily
* 🧑‍🤝‍🧑 Connect with nearby users
* ⚡ Smooth animations & premium UI
* 📱 Fully responsive mobile experience

---

## 🛠️ Tech Stack

* **Flutter** (Frontend)
* **Firebase Auth** (Authentication)
* **Cloud Firestore** (Database)
* **Firebase Storage** (Images)
* **Geolocator** (Location services)
